# ryu-s-wallet

## 프로젝트 개요

이더리움 테스트넷을 활용하여 블록체인 지갑생성 및 erc20 기본적인 핸들링 및 경험을 할수있는<br>안드로이드 앱 개발

프론트: react-native
백엔드: nestjs  
블록체인테스트넷: eth sepolia test
